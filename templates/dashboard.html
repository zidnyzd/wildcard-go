Panduan Setup

Jalankan script ini di VPS Ubuntu/Debian Anda:

ssh root@IP_VPS_ANDA
curl -sS -o vps-setup https://raw.githubusercontent.com/zidnyzd/wildcard-go/main/vps-setup && chmod +x vps-setup && sudo ./vps-setup
Script akan install Nginx, buat folder ACME challenge, konfigurasi firewall, dan tampilkan IP VPS Anda.

  1. Daftar di cloudflare.com (gratis)
  2. Klik Add a Site - masukkan domain Anda
  3. Pilih paket Free
  4. Update nameserver domain di registrar sesuai instruksi Cloudflare
  5. Tunggu hingga status domain Active

SSL for SaaS (Custom Hostnames) butuh payment method terverifikasi meskipun 100 hostname pertama gratis.

  1. Dashboard Cloudflare - Billing - Payment Methods
  2. Klik Add Payment Method (CC/PayPal)
  3. Cloudflare akan auth ~$1 (direfund otomatis)

  1. Profile - API Tokens - Create Custom Token
  2. Permissions: Zone:DNS:Edit + Zone:SSL:Edit
  3. Zone Resources: Include - Specific zone - pilih domain Anda
  4. Copy token (hanya ditampilkan sekali!)
  5. Zone ID: Dashboard domain - scroll kanan bawah

Menu Credentials - isi API Token, Zone ID, Domain, dan IP VPS. Klik Save & Verify.

Menu Setup Fallback - klik tombol Jalankan Setup Fallback. Ini akan membuat DNS records dan set fallback origin.

Menu Create Hostname - masukkan target domain (contoh: support.zoom.us). SSL akan aktif dalam 2-5 menit.
